void Inventory::findPotion ( Item *& potion,
int & index )

Finds the first potion in the inventory.

This method searches the inventory for the first item categorized as a "Potion". If potion is found, it sets the potion pointer to potion item and updates the index to the position of the potion in the inventory (m_items). If no potion is found, potion remains nullptr and index is set to -1. The potion found is used by the player to restore health in function drinkPotion.

Parameters
[out]potion- reference to a pointer to the potion found, or nullptr if no potion is available.
[out]index- index of the found potion in the inventory, or -1 if no potion is found.
