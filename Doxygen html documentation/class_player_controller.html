Detailed Description

Manages player's movement and interactions within the room.

The PlayerController class handles the movement of the player, including determining valid moves, updating the player's position on the map, and managing interactions with entities such as items, NPCs, and enemies. It also manages transitions between different rooms.

void PlayerController::createNextPosition ( )

Create the player's next position based on their current direction.

The method first gets the player's old position (getOldPosition()) to m_playerNextPosition and then modifies it based on the movement direction that was get in different method. The direction is provided as a character ('W', 'A', 'S', 'D'), corresponding to movement.

  • 'W' (up): Decreases the X coordinate
  • 'S' (down): Increases the X coordinate
  • 'A' (left): Decreases the Y coordinate
  • 'D' (right): Increases the Y coordinate
